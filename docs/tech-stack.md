# Technology Stack

## Backend
- **Framework**: Spring Boot 3.x
- **Language**: Java 17+
- **Build Tool**: Maven
- **Security**: Spring Security + JWT

## Database
- **Primary**: MySQL 8.x
- **ORM**: Spring Data JPA (Hibernate)

## Storage
- **Video Storage**: Local filesystem (mapped volume)
- **Future**: Consider S3-compatible storage

## Video Processing
- **Transcoding**: FFmpeg
- **Streaming**: HLS (HTTP Live Streaming)

## AI Features (Future)
- **Transcription**: Whisper API
- **Recommendations**: Vector embeddings + similarity search

## Frontend (Future Phase)
- React 18+
- Axios for API calls
- Video.js for player

## DevOps
- **Containerization**: Docker & Docker Compose
- **Version Control**: Git
- **CI/CD**: (TBD - GitHub Actions/Jenkins)
